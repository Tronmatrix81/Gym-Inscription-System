# Gym-Inscription-System
A basic gym security system to create and validate Gym credentials

Update: For the scanning function you must install these libraries (opencv, pyzbar):
    Windows: pip install opencv-python pyzbar
    Debian: sudo apt-get update \ sudo apt-get install libzbar0 \ pip install opencv-python pyzbar
    Bazzite: sudo dnf install zbar \ pip install opencv-python pyzbar
